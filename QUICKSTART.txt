QUICKSTART GUIDE:

== Windows (PowerShell) ==
1) Right-click `Start-Here-Windows.ps1` → "Run with PowerShell"
2) Log in when GitHub/Stripe/Firebase windows open
3) Paste secrets when prompted (you can skip and add later in GitHub → Settings → Secrets)
4) Script creates private repo, pushes code, sets up Firebase, creates Stripe products

== macOS/Linux ==
1) Open Terminal in this folder
2) `chmod +x Start-Here-Mac.sh && ./Start-Here-Mac.sh`

== Manual Setup ==
1) Install dependencies: `npm install`
2) Copy environment file: `cp .env.example .env`
3) Edit .env with your API keys:
   - OpenAI API key for oracle functionality
   - Stripe keys for billing
   - Firebase config for authentication
4) Start development: `npm run dev`
5) Open http://localhost:3000

The app includes:
- Oracle chat (requires OpenAI API key)
- User authentication (requires Firebase config)
- Subscription billing (requires Stripe config)
- Spiritual rituals library

If anything errors, check the exact error message and refer to the README.md for detailed setup instructions.
